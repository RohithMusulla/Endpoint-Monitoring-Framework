\# Endpoint Telemetry Behavior Analysis



Research-oriented multi-modal endpoint telemetry framework developed for cybersecurity behavioral analysis, concurrent systems evaluation, and resource-performance profiling.



\## Features



\* Keyboard event monitoring

\* Screenshot capture analysis

\* Webcam telemetry evaluation

\* Audio capture testing

\* Concurrent threading architecture

\* Resource utilization profiling

\* SMTP telemetry transmission testing



\## Research Focus



This project was developed strictly for academic and defensive cybersecurity research purposes in controlled laboratory environments.



The work focuses on:



\* Behavioral analysis

\* Endpoint telemetry

\* Concurrent systems performance

\* Resource efficiency

\* Detection fingerprint evaluation



\## Technologies Used



\* Python

\* OpenCV

\* Pillow

\* pynput

\* sounddevice

\* scipy
